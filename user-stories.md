# User Stories

1. As a user, I want to sign up so I can create an account.
2. As a user, I want to log in to access the app.
3. As a user, I want to view home content.
4. As a user, I want to view item details.
5. As a user, I want to save favorites.
6. As a user, I want offline local storage support.
7. As a user, I want API data shown in app.
8. As a user, I want settings control.
9. As a user, I want notifications.
